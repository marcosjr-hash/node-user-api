# Node User API

API REST desenvolvida com Node.js, Express, Prisma ORM e MongoDB.

O projeto foi criado com foco em praticar arquitetura backend, separação de responsabilidades e operações CRUD.

## Funcionalidades

- Criar usuários
- Listar usuários
- Atualizar usuários
- Remover usuários
- Filtro de usuários por nome, email e idade
- Tratamento básico de erros

## Tecnologias utilizadas

- Node.js
- Express
- Prisma ORM
- MongoDB

## Estrutura do projeto

```plaintext
src/
├── config/
├── controllers/
├── routes/
├── services/
```

## Conceitos praticados

- API REST
- Arquitetura em camadas
- CRUD
- Rotas HTTP
- Controllers e Services
- Integração com banco de dados
- Prisma ORM
- Tratamento de erros
